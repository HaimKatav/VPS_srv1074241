# TraderB Wiki

This vault is the **brain** of TraderB. See ../traderb/openclaw/TraderB_Wiki_Plan.md (or ~/Obsidian/TraderB/_plan.md after sync) for the operating model.

## Non-negotiable rules
1. **Wiki-first**: no code change without a wiki page describing the why.
2. **Provenance**: pages carry one of { extracted, inferred, ambiguous, approved-by-haim }. Trading rules must be approved-by-haim.

## Write discipline
- Only project-lead (Ari) / Librarian role writes unprompted.
- Other personas request a page via handoff, don't scribble directly.

## Layout (populated by /wiki-setup)
- _raw/             — staging, unprocessed captures
- _meta/taxonomy.md — canonical tag vocabulary
- _insights.md     — output of wiki analysis
- .manifest.json   — source ingestion tracking
- (concept pages)  — added as we ingest
